NequiBot Chatbot Integration Demo

Embedded Chatbot

The chatbot component is loaded from: https://web-components-dev.bancadigital.com.co/wc-chatbot-mas/wc-main-mas.js

Alternative Configuration

Example with custom user data and different provider:

<wc-chatbot web="true" provider="NEQUI_NEGOCIOS" user='{"name": "Test User", "phoneNumber": "3000000000", "email": "test@example.com"}' position="banner"> </wc-chatbot>

Usage Instructions

  1. The web component is automatically loaded from the CDN
  2. Use the <wc-chatbot> tag to embed the chatbot
  3. Configure the component using attributes:
    • web: "true" for web interface, "false" for mobile
    • provider: "NEQUI_PERSONAL" or "NEQUI_NEGOCIOS"
    • position: "fixed", "banner", etc.
    • user: JSON string with user information (optional)
  4. The chatbot will appear as a floating button that users can click to start a conversation
